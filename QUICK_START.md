## ⚡ 5-Minute Quick Start

### Step 1: Test Locally (1 minute)
Simply open `index.html` in your browser. No server needed!

```
File → Open → Select index.html
```

Website should load instantly with all styles applied.

### Step 2: Deploy to Netlify (2 minutes)

1. **Create account**: https://netlify.com (free)
2. **Drag & drop** `index.html` and `contact.html` into Netlify
3. **Done!** Your site is live with instant HTTPS ✨

**Your site will be at**: `randomname.netlify.app`

### Step 3: Connect Custom Domain (24 hours)

1. **Buy domain**: `edizonecareers.com` (~$12/year)
2. **In Netlify**: Settings → Domain Management → Add Custom Domain
3. **Follow DNS instructions** (usually 24-48 hours to activate)
4. **Done!** You're live on your own domain

Open `index.html` in any text editor (VS Code, Notepad++, etc.) and search for:
