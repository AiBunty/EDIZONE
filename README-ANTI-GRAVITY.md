## 🎯 Feature Details

### **1. Floating Hero Card**
```
Animation: 8-second vertical float with subtle rotation
Purpose: Draws eye to key content, creates sense of movement
Use Case: Profile highlights, featured services
```

### **2. Parallax Scrolling** 
```
Effect: Background glows move slower than content
Purpose: Creates depth, enhances immersion
Speed: GSAP scrub = 1 (adjust in JS)
```

### **3. Magnetic Buttons**
```
Trigger: Cursor movement near button
Response: Button follows cursor (0.3 multiplier)
Reset: Smooth return on mouse leave
Effect: Increases perceived interactivity
```

### **4. Bento Grid Layout**
```
Grid: CSS auto-fit, minmax(250px, 1fr)
Gap: 24px spacing
Responsive: Single column on mobile, multi-column on desktop
Services: 12 service boxes with auto-wrap
```

### **5. 3D Radar Chart (GPATHS)**
```
Components: 4 concentric rings + 5 interactive nodes
Animation: Continuous 20-second rotation
Interaction: Nodes glow on hover with scale effect
Symbolism: Interconnected career assessment framework
```

### **6. Scroll Animations**
```
Trigger: Intersection Observer (10% visibility)
Animation: fadeInUp (0.6 seconds)
Elements: All service cards, reviews, testimonials
Effect: Progressive disclosure as user scrolls
```

### **7. Trust Dock Badge**
```
Position: Fixed bottom-left corner
Visibility: Always visible
Purpose: Shows DCore verification
Elements: LLPIN, PAN, certification links
Animation: Slides in from left on load
```
